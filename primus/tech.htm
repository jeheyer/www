This page was developed mainly on a Micron Millenia Pro. I'd recommend Micron to anyone looking for a high quality, professional computer system with great tech support. My system has a Diamond Stealth 3D 2000 video card and a SoundBlaster AWE 64 sound card. For information about my computer can be found here.

For capturing video, I used a Power Macintosh 7500 using Apple Video Player and Avid VideoShop.

On the Software side, my computer runs Windows NT and Linux. To create HTML and VRML, I used Ultra-Edit 32. For creating graphics, I used a combination of PhotoShop 4.0 and Paint Shop Pro. Sound recording was done with Creative WaveStudio, and CoolEdit was used as a converter. For Java Applets, I used Sun JDK.
