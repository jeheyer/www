Primus - Bob

Tabbed by Christopher Preston. 

This is played on a 6 string frettless, but I've figured it out on a 4
string fretless.  It can be done on a fretted, but it doesn't sound as
good. Drop the "E" to "D", and finger each note, until the
intro/verseriff, which is slapped.  The chorus is fingernail strummed,
with the low note slapped.
