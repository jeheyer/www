The Cheesy Primus Ring

If you have a Primus page and are interested in joining the Ring, go to Ram's information page and fill out the form. Once you're in the queue, add the appropriate ring code to your page. Check below for the various code you can use (make sure you replace 'XX' with your real webring ID). Once you've added the code, E-Mail someone so they can add you to the Ring.
